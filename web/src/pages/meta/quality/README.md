# 元数据质量页面

## 功能概述

元数据质量页面提供了对数据库元数据质量的全面监控和分析功能，帮助用户了解元数据的完整性和准确性。

## 页面结构

### 上部分：概览数据统计
- **数据库总数**：显示系统中的数据库总数量
- **数据表总数**：显示系统中的数据表总数量  
- **数据字段总数**：显示系统中的数据字段总数量

### 中部分：质量指标
- **数据库业务关联率**：显示已关联业务的数据库占比
- **数据表注释完备率**：显示有注释的数据表占比
- **数据字段注释完备率**：显示有注释的数据字段占比
- **数据表备注准确度**：显示数据表备注的准确度评估
- **数据字段备注准确度**：显示数据字段备注的准确度评估

### 下部分：图表展示
- **数据库业务关联情况**：饼图展示已关联和未关联业务的数据库分布
- **数据表注释完备情况**：饼图展示有注释和无注释的数据表分布
- **数据字段注释完备情况**：饼图展示有注释和无注释的数据字段分布
- **表注释准确度分布**：饼图展示表注释准确度（准确、一般、不准确）的分布
- **字段注释准确度分布**：饼图展示字段注释准确度（准确、一般、不准确）的分布

### 注释准确度判断规则
- **准确**：包含3个及以上中文字符的注释
- **一般**：包含1-2个中文字符的注释
- **不准确**：纯英文或无中文字符的注释

## 质量等级说明

- **绿色（≥80%）**：质量优秀
- **黄色（60%-80%）**：质量良好，需要改进
- **红色（<60%）**：质量较差，需要重点关注

## 技术实现

- 使用 Ant Design Pro 组件库
- 使用 @ant-design/plots 图表库
- 响应式设计，支持移动端访问
- 模拟数据展示，可对接真实后端API
